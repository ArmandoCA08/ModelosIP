El repositorio contiene una aplicación desarrollada en **MATLAB App Designer**, donde se construyen curvas de productividad para diferentes modelos de afluencia.

## Vista previa de modelos IPR

![Comportamiento de afluencia de modelos IPR](Im%C3%A1genes/Comportamiento%20de%20afluencia%20de%20modelos%20IPR.png)

## Modelos incluidos

- Moore
- Vogel
- Standing
- Harrison
- Vogel generalizado

Si desea consultar de forma independiente cada modelo, sus ecuaciones, citas y algoritmos, podrá encontrarlos en la carpeta **Modelos** de este repositorio.
