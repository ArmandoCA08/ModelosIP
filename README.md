## Vista previa de modelos IP

![Comportamiento de afluencia de modelos IP](Im%C3%A1genes/Comportamiento%20de%20afluencia%20de%20modelos%20IP.png)

## Modelos incluidos

- Moore
- Vogel
- Standing
- Harrison
- Vogel generalizado

Si desea consultar de forma independiente cada modelo, sus escuaciones, citas y algoritmos podrá encontrarlas en la carpeta Modelos de este repositorio.
